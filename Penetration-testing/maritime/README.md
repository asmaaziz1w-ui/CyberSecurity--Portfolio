Maritime Navigation System -Penetration Test
A Penetration testing Assessment of A simulated maritime navigation Platform by NEOM Smart City (Vision 2030).
Key Facts:
- 20 Vulnerabilities identified (8 Critical, 10 High,)
- Tools: Nmap, Wireshark, Metasploit, Burp Suite
- Framework: PTES+ OWASP + NIST SP 800-115

  Systems Assessed
  - AIS (Automatic Identification System)
  - VTS (Vessel Traffic Services) Dashboard
  - Port Management Web Application
  - SCADA & IOT Marine Sensors

  - < ALL Testing conducted in an isolated Lab Environment For Academic Purposes.
